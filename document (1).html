דוקומנטציה

דף זה מיועד להסבר טכני על המערכת Sentramind Turret: ארכיטקטורה, חיבורים, מסדי נתונים, ועוד. אתה יכול להעתיק מפה חלקים למסמך רשמי (Word / PDF) של הפרויקט.

ארכיטקטורה כללית

  • מיקרו־בקר: ESP32 DevKit
  • חיישן מרחק: HC-SR04 (distanceCm)
  • חיישן טמפ׳/לחות: AHT10 (לא חובה ברדאר, אבל אפשר)
  • בסיס נתונים: Firebase Realtime Database
  • Frontend: HTML + Bootstrap + JavaScript

מבנה מסד נתונים (דוגמא)

        
radar/

  currentAngle: number

חישוב מיקום נקודה ברדאר

ב־radar.html אתה מחשב את מיקום הנקודה לפי זווית הצריח והמרחק. 

        
const radarSize = 260;
const maxDist   = 200;

const norm = Math.min(dist, maxDist) / maxDist;
const r    = (radarSize / 2) * norm;

const angleRad = (currentAngleDeg - 90) * Math.PI / 180;

const cx = radarSize / 2;
const cy = radarSize / 2;
const x = cx + r * Math.cos(angleRad);
const y = cy + r * Math.sin(angleRad);

את החלק הזה אפשר להסביר במסמך הרשמי: איך מתרגמים נתוני חיישן מרחק وزווית לקואורדינטות X/Y על המסך.
